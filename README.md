# Agent optymalizacyjny do rozwiązywania problemu komiwojażera (TSP)

Projekt w Pythonie służący do rozwiązywania problemu komiwojażera z wykorzystaniem algorytmów optymalizacyjnych. 

## Funkcjonalności
- Algorytm rozwiązujący problem TSP
- Interfejs do wprowadzania danych (lista miast i odległości)
- Wizualizacja trasy

## Uruchomienie
1. Zainstaluj wymagane zależności: `pip install -r requirements.txt`
2. Uruchom główny plik projektu: `python main.py`

## Wymagane rozszerzenia VS Code
- Python
- Python Environment Manager
